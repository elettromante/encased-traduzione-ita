# Changelog - EncasedBoy (fork di Encased 1.3.1517.1645 di DocteurKain)



## [1.1.0] - 25-02-2026

### Aggiunto
- **Parametro `--extract_json`**: Implementato il flag per l'estrazione manuale. Ora l'operazione non è più automatica, eliminando il rischio di sovrascrittura distruttiva del lavoro.
- **Logica di Compilazione**: Attivato il metodo `LocaleToFile`. Il tool ora genera il file binario `IT.locale` partendo dal JSON tradotto.

### Modificato
- **Flusso di lavoro**: Il programma esegue ora la generazione del file di gioco come operazione predefinita (default).
- **Allineamento Framework**: Aggiornati i file `.csproj` per risolvere l'errore di compatibilità NU1201 tra i progetti.
- **Gestione File**: Inseriti controlli di sicurezza `File.Exists` per impedire la perdita accidentale di dati nel file `En_Traduzione.json`.
